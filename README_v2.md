## Quickstart

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run ETL (loads CSVs → SQLite, creates views, runs simulations)
cd pricing_project_v2
python python/etl_pipeline.py

# 3. Run full analysis (7 modules, 7 charts saved to /data/)
python python/pricing_analysis.py

# 4. Run unit tests
pytest tests/ -v

# 5. Open dashboard
open pricing_strategy_dashboard_v2.html
```

## SQL Highlights

- **Window functions**: `RANK()`, `PERCENT_RANK()`, `LAG()`, `LEAD()`, `FIRST_VALUE()`, rolling averages
- **CTEs**: Multi-step elasticity calculation, price extreme detection
- **Stored procedure**: `sp_discount_impact` — pass category + discount → get revenue projection + recommendation
- **SCD Type 2**: `price_history` table with `valid_from`/`valid_to` columns
- **Pivot table**: Revenue by Category × Price Tier
- **Cohort analysis**: Monthly revenue trend with cumulative window sum
